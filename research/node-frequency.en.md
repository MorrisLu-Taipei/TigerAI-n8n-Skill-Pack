## 5. AI must remember "boring but essential" nodes

High frequency but easily overlooked by beginners:

| Node | Use | When AI must auto-add |
|---|---|---|
| `set` | shape data | cross-node field transformation |
| `noOp` | branch converge | want to merge after if/switch |
| `stopAndError` | manual error throw | validation fails |
| `merge` | multi-input merge | parallel branch ends |
| `splitOut` | array → single items | upstream returns array, downstream needs single |
| `aggregate` | collect batch results | after splitInBatches, need to combine |
| `respondToWebhook` | webhook response | webhook trigger requires it |
